# AMA Answers

1: **Difference Between Inline and Block Elements in HTML:**  
Inline elements only take up as much width as necessary and do not start on a new line (e.g., `<span>`, `<a>`). Block elements take the full width available and always start on a new line (e.g., `<div>`, `<p>`).

---

2: **What is HTML?**  
HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages using elements and tags.

---

3: **Attribute for Showing Alt Text in HTML Image:**  
The `alt` attribute in the `<img>` tag is used to display alternative text if the image cannot be loaded.

Example:
```html
<img src="image.jpg" alt="Sample Image">
```

---

4: **Priority Between Inline, Internal, and External CSS:**  
CSS priority order is:

1. Inline CSS (Highest)
2. Internal CSS
3. External CSS (Lowest)

---

5: **What is CSS?**  
CSS (Cascading Style Sheets) is used to style and design HTML elements such as colors, fonts, spacing, layouts, and responsiveness.

---

6: **Difference Between ID and Class in HTML:**  

- `id` is unique and used for one element only.
- `class` can be used for multiple elements.

---

7: **Difference Between Div and Span:**  

- `<div>` is a block-level element.
- `<span>` is an inline element.

`<div>` is mainly used for large sections, while `<span>` is used for small inline styling.

---

8: **Difference Between Div and Section:**  

- `<div>` is a generic container without semantic meaning.
- `<section>` is a semantic tag used to define meaningful sections of content.

---

9: **Different Ways of Using CSS:**  

There are 3 ways to use CSS:

1. Inline CSS
2. Internal CSS
3. External CSS

---

10: **How Can We Connect an External CSS File?**  

Using the `<link>` tag inside the `<head>` section.

Example:
```html
<link rel="stylesheet" href="style.css">
```

---

11: **Use of `<br>` Tag:**  

The `<br>` tag is used to insert a line break in HTML.

---

12: **What are Inline-Block Elements in HTML?**  

Inline-block elements behave like inline elements but allow width and height properties like block elements.

---

13: **Difference Between Tag and Attribute:**  

- A tag defines an HTML element.
- An attribute provides additional information about the element.

---

14: **Why Do We Use Media Queries?**  

Media queries are used to make websites responsive for different screen sizes such as mobile, tablet, and desktop.

---

15: **What is ON in SQL?**  

The `ON` keyword is used with joins to specify the condition for matching rows between tables.
